What you can find below is an evolution simulation. Each generation, 30 individuals are breeded from the 20 individuals who scored the best.

The goal for each creature is to propulse itself to the targets, in the order they're presented.

For that, a creature can propulse itself once per target. Each propulsion has an angle, a force, a time it sets off (from the last target) and a duration. These four variables make up the alleles of one gene. The number of genes increases with the number of targets passed.

At the 10th target, the wall are set on fire and creatures cannot bounce on them anymore. This usually hardens the task A LOT.
